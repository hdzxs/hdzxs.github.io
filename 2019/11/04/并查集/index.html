并查集

Nov 4, 2019 #study

  这个合根植物问题是一个并查集问题，即寻找一个集合的子集之间的关系。

问题重述

w星球的一个种植园，被分成 m * n 个小格子（东西方向m行，南北方向n列）。每个格子里种了一株合根植物。这种植物有个特点，它的根可能会沿着南北或东西方向伸展，从而与另一个格子的植物合成为一体。

如果我们告诉你哪些小格子间出现了连根现象，你能说出这个园中一共有多少株合根植物吗？

输入格式

第一行，两个整数m，n，用空格分开，表示格子的行数、列数（1<m,n<1000）。 　　

接下来一行，一个整数k，表示下面还有k行数据(0<k<100000) 　

接下来k行，第行两个整数a，b，表示编号为a的小格子和编号为b的小格子合根了。格子的编号一行一行，从上到下，从左到右编号。比如：5 * 4 的小格子，编号：1到20. 　　

样例输入：

5 4

16

2 3

1 5

5 9

4 8

7 8

9 10

10 11

11 12

10 14

12 16

14 18

17 18

15 19

19 20

9 13

合根情况

问题思路：并查集

是用来解决一个大集合中子集之间关系的问题，比如说有一个大集合S，元素为{a b c d e}。他的子集包括S1{a,b} S2{c,d}等集合，现任意给两个元素，判断是否在一个集合中；如果我们合并了一些集合，结果又会如何。并查集就是在解决这个问题。

关于并查集的知识，这里贴出一个讲的很清楚的博客的链接，代码本身很好懂，但是思想很精髓。

并查集及C++实现与优化

解题代码

#include<iostream>
#include<vector>
using namespace std;

class DisjSet
{
private:
	int parent[1000000];

public:
	DisjSet(int max_size)
	{
		// 初始化每一个元素的根节点都为自身
		for (int i = 1; i <= max_size; ++i)
			parent[i] = i;
	}
	int find(int x)
	{
		return parent[x] == x ? x : find(parent[x]);
	}
	void to_union(int x1, int x2)
	{
		if(find(x1) != find(x2))
			parent[find(x1)] = find(x2);
	}
	// 判断两个元素是否属于同一个集合
	bool is_same(int e1, int e2)
	{
		return find(e1) == find(e2);
	}
};

int main() {
	int m, n = 0;
	int vis[1000000] = { 0 };
	int temp;
	int ans = 0;

	cin >> m >> n;
	cin >> temp;
	
	int a, b = 0;

	DisjSet disjSet = DisjSet(m*n);
	for (int i = 0; i < temp; i++) {
		cin >> a >> b;
		disjSet.to_union(a,b);
	}
	
	for (int i = 1; i <= m*n; i++) {
		vis[disjSet.find(i)] = 1;
	}

	for (int i = 1; i <= m*n; i++) {
		ans += vis[i];
	}
	cout << ans << endl;
	return 0;
}

代码还是比较简单的，一开始范围错了出了问题，主要还是并查集的思想比较重要吧。
