自学java之路＋随笔

Nov 12, 2019 #essay #study

  开始自学JAVA，为什么？学了四年计算机JAVA都不会太low了吧，仅此而已。然后，我想顺便再记下最近的事吧。

JAVA的HelloWorld

感觉就像背单词每次都背第一个一样，学语言可能都是helloworld吧。

因为以前的基础，前面这点看起来没啥难的，一会就看完了。可能就是JDK、JRE的安装费点劲，不过之前也弄过很久了，所以也没卡住。等什么时候学到JAVA的精髓再来写写吧。

最近的事

感觉最近干什么了呢？好像还是挺忙的，软件终于快结课了（有机会一定会把这个小项目记下来）；英语看的是有点少，不过还是有在看的；蓝桥杯好像是没咋做了，感觉有点简单；还有srdp的事，以后可能会在这说一下整个项目吧（给自己留了第二个坑。。。）。

现在的想法是把JAVA认真学一学，然后想找个好点的github项目做一做，基本就这样吧。

不过话说没有几周就是期末考了，今年成绩还是十分十分重要的！！（突然发现原来保研变成15%了？！）小硕要加油啊！

心情

怎么说呢？最近总有种在重复生活的感觉。说不上烦，就是觉得自己是不是有点太不爱出门了？好像缺少点真正追求的目标？是我追求的太简单了吗？现在能想到的就是想保研吧，可是现在我该干嘛呢？最近在做的事真的对自己有帮助吗？最初想做这些是想提升自己，但是总觉得，哪里做错了，心理这种不爽的感觉还是挥之不去。感觉现在的自己就很奇怪，会因为自己起来的晚而跟自己生气，但是，真的起来了又不知道该做什么。感觉自己快成为那种把一年活成一天的人了。可是我最近在做的改变的目的又是为啥呢？？？

一说就说的多了，其实也不知道是在说给谁听，也不知道自己这样写下来又能有什么帮助。就像我不知道我买块键盘，能写多少代码一样。只是想写，只是想买，就这样吧。

小硕加油吧，要努力，要把生活活的更加多姿多彩！

Cooper

Student in Computer Sicence，OUC
