Hugo添加归档页面

Oct 30, 2019 #study #hugo

  刚刚参照网上的思路，为hugo添加了归档页面，这里做个整理。

1.创建归档页面模板

  1.在hugo根目录下新建模板文件(./layout/archive/signal.html)

  2.将主题下的默认模板内容复制过来，网上的是复制了signal模板，这个是单个页面的显示模板，我这里觉得list模板更好看，就复制了list的内容。

2.替换页面展示部分代码

  将新建的模板文件中的内容展示部分代码换成下面这个（页面展示部分一般就是content标签内的部分）：

<h2>
{{ range where .Site.Pages "Section" "post" }}
{{ range (.Pages.GroupByDate "2006") }}
<details class="menu" open>
<summary>{{ .Key }}</summary>
{{ range .Pages }}
<article><h3><a href="{{ .Permalink }}" title="{{ .Title }}">{{ .Title }}</a></h3></article>
{{ end }}
</details>
{{ end }}
{{ end }}
</h2>

  • .Site.Pages决定了归档目录，这里是归档了/content/post文件夹。

  • .Pages.GroupByDate，根据日期归档，“2006”，按照年份归档；2006-01，按照年月归档。

3.新建归档页面

  1. 新建一个归档页面./content/archive.md

  2. 在文件头添加type字段来引用归档页面模板

  3. 在文件头添加menu字段将归档页面链接添加到主目录

    ---
title: "Archive"
type: "archive"
menu: "main"
name: Archive
weight: 1
---
