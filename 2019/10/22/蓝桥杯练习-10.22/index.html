数字超出范围

#include<stdio.h>
#include<iostream>
using namespace std;

int main() {
	int f1 = 1;
	int f2 = 1;
	int f3 = 1;

	int n = 0;
	cin >> n;

	for (int i = 0; i < n - 2; i++) {
		f3 = (f1 + f2)%10007;
		f1 = f2;
		f2 = f3;
	}

	int outcome = f3 % 10007;

	cout << outcome;

	return 0;
}

  计算机组成原理还是挺重要的，虽然写的时候还是没反应过来，不过之后还是能理解的，不断地大数叠加查过了int的范围，出现了负数，所以这里每次只保存余数。
