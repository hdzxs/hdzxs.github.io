PATA_A_1097

Apr 25, 2020 #study

PAT_A_1097

Given a singly linked list L with integer keys, you are supposed to remove the nodes with duplicated absolute values of the keys. That is, for each value K, only the first node of which the value or absolute value of its key equals K will be kept. At the mean time, all the removed nodes must be kept in a separate list. For example, given L being 21→-15→-15→-7→15, you must output 21→-15→-7, and the removed list -15→15.

Input Specification:

Each input file contains one test case. For each case, the first line contains the address of the first node, and a positive N (≤105) which is the total number of nodes. The address of a node is a 5-digit nonnegative integer, and NULL is represented by −1.

Then N lines follow, each describes a node in the format:

Address Key Next

where Address is the position of the node, Key is an integer of which absolute value is no more than 104, and Next is the position of the next node.

Output Specification:

For each case, output the resulting linked list first, then the removed list. Each node occupies a line, and is printed in the same format as in the input.

Sample Input:

00100 5
99999 -7 87654
23854 -15 00000
87654 15 -1
00000 -15 99999
00100 21 23854

Sample Output:

00100 21 23854
23854 -15 99999
99999 -7 -1
00000 -15 87654
87654 15 -1

问题分析

链表问题，首先创建一个静态链表，创建一个数组用来判断是否已存在。读入数据之后，从start开始遍历，对于首次发现的存入ans链表，对于已存在的存入del链表（因为题目要求输出被删除结点）。

注意事项

  • 注意数组大小，每次总开一样大小的数组，以后要大一点，否则对于数据量比较大的会有问题。
  • 静态链表存储的时候按照节点存储，但是最后输出可以从零开始，只需要把每个节点的地址也保存进去node中，而不是作为真正的数组地址。
  • 书中给出的解决方案只利用了一个链表，利用maxn作为分界线，将无效结点作为2*maxn。我因为分成了多个链表所以不需要考虑无效结点的问题。如果是这种做法，需要在node中添加一个flag来说代表无效结点。
  • 注意输出格式，最后一个节点要输出-1，注意要输出五位数，利用printf更加简单。

代码实现

#include<iostream>
#include<algorithm>
#include<math.h>
using namespace std;

struct Node{
	int next;
	int data;
	int address;
}nodes[100022],ans[100022],delnodes[100022];

int pd[10002] = { 0 };

int main() {
	int start, n;
	int delcount = 0, count = 0;
	int index = 0, delindex = 0;
	int num = 0;                            //暂时的绝对值

	cin >> start >> n;

	int address, data, next;
	for (int i = 0; i < n; i++) {
		
		cin >> address >> data >> next;

		nodes[address].address = address;
		nodes[address].data = data;
		nodes[address].next = next;

	}

	for (int i = start; i != -1; i = nodes[i].next) {
		
		num = abs(nodes[i].data);
		
		if (pd[num] == 0) {
			//cout << "asdasd"<<num;
			ans[index] = nodes[i];
			
			if (index != 0)
				ans[index - 1].next = ans[index].address;

			index++;
			count++;
			pd[num] = 1;
		}
		else { //删除掉结点，并将其存储起来
			delnodes[delindex] = nodes[i];

			if (delindex != 0)
				delnodes[delindex - 1].next = delnodes[delindex].address;

			delindex++;
			delcount++;				//记录删除掉的结点数
		}
				
	}

	for (int i = 0; i <count; i++) {

		printf("%05d ", ans[i].address);
		printf("%d ", ans[i].data);
		if (i != count - 1)
			printf("%05d\n", ans[i].next);
		else
			printf("%d\n", -1);
		
	}

	for (int i = 0; i < delcount; i++) {

		printf("%05d ", delnodes[i].address);
		printf("%d ", delnodes[i].data);
		if (i != delcount - 1)
			printf("%05d\n", delnodes[i].next);
		else
			printf("%d\n", -1);
	}

	return 0;

}
