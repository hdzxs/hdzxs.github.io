Apr 14, 2020 #study

PAT_A_1052

A linked list consists of a series of structures, which are not necessarily adjacent in memory. We assume that each structure contains an integer key and a Next pointer to the next structure. Now given a linked list, you are supposed to sort the structures according to their key values in increasing order.

Input Specification:

Each input file contains one test case. For each case, the first line contains a positive N (<105) and an address of the head node, where N is the total number of nodes in memory and the address of a node is a 5-digit positive integer. NULL is represented by −1.

Then N lines follow, each describes a node in the format:

Address Key Next

where Address is the address of the node in memory, Key is an integer in [−105,105], and Next is the address of the next node. It is guaranteed that all the keys are distinct and there is no cycle in the linked list starting from the head node.

Output Specification:

For each test case, the output format is the same as that of the input, where N is the total number of nodes in the list and all the nodes must be sorted order.

Sample Input:

5 00001
11111 100 -1
00001 0 22222
33333 100000 11111
12345 -1 33333
22222 1000 12345

Sample Output:

5 12345
12345 -1 00001
00001 0 11111
11111 100 22222
22222 1000 33333
33333 100000 -1

问题分析

该题目地址较短，使用今天学习的静态链表解决。开局设置Node表示链表中的点，注意之后会对其排序所以node中药包含add。同时设置flag来表示该点是否有效，之后通过排序抛弃无效点。对于有效的点按照题目要求排序，最后从头输出即可。

注意事项

  • 排序之后将无效的点排到了后面有效的点在前面，这时候node的地址已经不是最开始输入的地址了，而是1、2、3等，所以需要提前将地址保存下来。
  • 输出时注意格式控制，还是prinf比较简单。
  • 因为是在排序时筛选是否有效，所以需要排序的范围是maxn。

代码实现

#include<iostream>
#include<algorithm>
#include<stdio.h>
using namespace std;


struct Node{
	int add;
	int data;
	int next;
	bool flag;
}node[100000];

bool cmp(Node a, Node b) {
	if (a.flag == false || b.flag == false) 
		//如果有一个结点是无效的，则将其放到后面
		return a.flag > b.flag;
	else
		return a.data < b.data;
}

int main() {
	int n, start;
	cin >> n >> start;
	int add, data, next;
	int count = 0;

	for (int i = 0; i < 100000; i++)
		node[i].flag = false;

	for (int i = 0; i < n; i++) {
		cin >> add >> data >> next;
		
		node[add].add = add;
		node[add].data = data;
		node[add].next = next;
	}

	for (int i = start; i != -1; i = node[i].next) {
		node[i].flag = true;
		count++;
	}

	sort(node, node + 100000, cmp);

	if (count == 0)
		cout << "0 -1";
	else {
		printf("%d %05d\n", count,node[0].add);;
		for (int i = 0; i < count; i++) {
			
			printf("%05d", node[i].add);

			cout << " " << node[i].data << " ";

			if (i == count - 1)
				printf("-1");
			else
				printf("%05d\n", node[i+1].add);
		}
	}

	return 0;
}
