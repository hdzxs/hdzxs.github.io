Feb 20, 2020 #study

PAT_A_1089

According to Wikipedia:

Insertion sort iterates, consuming one input element each repetition, and growing a sorted output list. Each iteration, insertion sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there. It repeats until no input elements remain.

Merge sort works as follows: Divide the unsorted list into N sublists, each containing 1 element (a list of 1 element is considered sorted). Then repeatedly merge two adjacent sublists to produce new sorted sublists until there is only 1 sublist remaining.

Now given the initial sequence of integers, together with a sequence which is a result of several iterations of some sorting method, can you tell which sorting method we are using?

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer N (≤100). Then in the next line, N integers are given as the initial sequence. The last line contains the partially sorted sequence of the N numbers. It is assumed that the target sequence is always ascending. All the numbers in a line are separated by a space.

Output Specification:

For each test case, print in the first line either “Insertion Sort” or “Merge Sort” to indicate the method used to obtain the partial result. Then run this method for one more iteration and output in the second line the resuling sequence. It is guaranteed that the answer is unique for each test case. All the numbers in a line must be separated by a space, and there must be no extra space at the end of the line.

Sample Input 1:

10
3 1 2 8 7 5 9 4 6 0
1 2 3 7 8 5 9 4 6 0

Sample Output 1:

Insertion Sort
1 2 3 5 7 8 9 4 6 0

Sample Input 2:

10
3 1 2 8 7 5 9 4 0 6
1 3 2 8 5 7 4 9 0 6

Sample Output 2:

Merge Sort
1 2 3 8 4 5 7 9 0 6

问题分析

这题的思路本身比较简单粗暴，对整个数组分别进行插入排序和归并排序，然后进行比较，遇到相等的就退出。于是重点就放在了如何进行插入和归并排序。

插入排序

插入排序，从头至尾每次选择一个数字将他插入前面的有序数据的正确位置之中，注意每次要讲要插入的数据保存下来。代码实现如下

 7     for (i = 1; i < length; i++)
 8     {
 9         pos = i -1 ;    //有序序列的最后一个元素位置
10         cur = num[i];    //保存待排序元素的值

归并排序

将数组每次分为几个小部分，然后两两归并为一个有序的数组，可以使用迭代和递归两种方式，该题里使用迭代方式,下面是递归版本：

void merge(int a[100], int left1, int right1, int left2, int right2) {
	int i = left1, j = left2;
	int temp[100],index = 0;
	
	while (i < right1 && j < right2) {
		if (a[i] <= a[j])
			temp[index++] = a[i++];
		else
			temp[index++] = a[j++];
	}

	while (i <= right1)
		temp[index++] = a[i++];
	while (j <= right2)
		temp[index++] = a[j++];

	for (int i = 0; i < index; i++)
		a[left1 + i] = temp[i];
	
}

void mergeSort(int a[100], int left, int right) {
	if (left < right) {
		int mid = (left + right) / 2;
		mergeSort(a, left, mid);
		mergeSort(a, mid + 1, right);
		merge(a, left, mid, mid + 1, right);
	}
}

注意事项

  1. 要将数组暂时保存下来，保证两个排序使用的都是原始无序数组。
  2. 归并排序这里使用迭代方式较为合适，并且因为数据量小，每次需要比较排序结果，所以可以使用sort函数来进行排序。

代码实现

#include<iostream>
#include<algorithm>
using namespace std;

bool isSame(int a[100], int b[100], int n) {
	for (int i = 0; i < n; i++)
		if (a[i] != b[i])
			return false;
	return true;
}

int main() {
	int n;
	cin >> n;
	int a[100], b[100],temps[100];

	for (int i = 0; i < n; i++) {
		cin >> a[i];
		temps[i] = a[i];
	}

	for (int i = 0; i < n; i++)
		cin >> b[i];

	bool isInsert = false;
	bool isMerge = false;
	//插入排序
	for (int i = 1; i < n; i++) {
		int j = i;
		//下面为插入排序部分，将当前需要插入的数据拿出来，暂存，每次与它比较，不断后移，比较出一个比他小的停止。
		int temp = temps[i];

		while (j > 0 && temps[j - 1] > temp) {
			temps[j] = temps[j - 1];
			j--;
		}
		temps[j] = temp;

		if (isInsert) {                                   //如果是插入排序，再排序一次，然后输出
			for (int k = 0; k < n - 1; k++)
				cout << temps[k] << " ";
			cout << temps[n - 1];
			return 0;
		}

		if (isSame(temps, b, n)) {                            
			cout << "Insertion Sort" << endl;
			isInsert = true;
		}
	}

	//归并排序
	//有两种实现方式，一种是递归，一种是迭代，下面迭代实现
	for (int i = 0; i < n; i++)
		temps[i] = a[i];
	
    //下为归并排序部分
	for(int step = 2; step/2 <= n; step *= 2){
		for (int i = 0; i < n; i += step) {
			sort(temps + i, temps + min(i + step, n));
		}

		if (isMerge) {
			for (int k = 0; k < n - 1; k++)
				cout << temps[k] << " ";
			cout << temps[n - 1];
			return 0;
		}

		if (isSame(temps, b, n)) {
			cout << "Merge Sort" << endl;
			isMerge = true;
		}
	}
}
