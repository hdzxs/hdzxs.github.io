PAT-A1082

Feb 8, 2020 #study

PATA1082

中国加油！武汉加油！

Given an integer with no more than 9 digits, you are supposed to read it in the traditional Chinese way. Output Fu first if it is negative. For example, -123456789 is read as Fu yi Yi er Qian san Bai si Shi wu Wan liu Qian qi Bai ba Shi jiu. Note: zero (ling) must be handled correctly according to the Chinese tradition. For example, 100800 is yi Shi Wan ling ba Bai.

Input Specification:

Each input file contains one test case, which gives an integer with no more than 9 digits.

Output Specification:

For each test case, print in a line the Chinese way of reading the number. The characters are separated by a space and there must be no extra space at the end of the line.

Sample Input 1:

-123456789

Sample Output 1:

Fu yi Yi er Qian san Bai si Shi wu Wan liu Qian qi Bai ba Shi jiu

Sample Input 2:

100800

Sample Output 2:

yi Shi Wan ling ba Bai

题目思路：

首先这个题要明白中文念数字是怎么念的，四位为一组，后面加单位。所以我们就可以将整个数字分为四个四个的一组，组内采用相同的手法来读，组外读额外的单位。

需要注意的问题：

  1. 0的问题，比如1008,1000008，要考虑是否有积蓄的0，如果有就要读出来；如果一直是0，比如100，就不需要读出来。
  2. 如果整个节一个被读出来的没有，注意不要读出单位。比如800000008应该读作“捌亿零捌”而不是“捌亿万零捌”，所以要加一个参数表示这个小节是否被读出来过。

解决办法：

添加left、right变量，用来指示一节的左右边界，left不断靠近right，相等之后移向下一节，直到结束。

代码实现：

#include<iostream>
#include<string>
using namespace std;

int main() {
	string num;
	string ans[20];
	int length = 0;

	cin >> num;

	string digit[10]{ "ling","yi","er","san","si","wu","liu","qi","ba","jiu" };
	string dan[5]{"Shi","Bai","Qian","Wan","Yi", };
	
	int left = 0, right = num.length() - 1;

	if (num[0] == '-') {
		cout << "Fu";
		left++;
	}

	if (num.length() == 1)
		cout << digit[num[0] - '0'];

	while (right >= left + 4) {
		right -= 4;
	}

	while (left < num.length()) {
		bool flag = false; //表示没有积蓄的0
		bool isPrint = false; //表示这一节是否输出过一位
		while(left <= right) {
			//如果当前为是0
			if (num[left] == '0') {
				flag = true;
			}
			//如果当前为不是0
			else {
				if (flag == true) {
					cout << " ling";
					flag = false;
				}

				if (left > 0)
					cout << " ";
				
				cout << digit[num[left] - '0'];
				isPrint = true;

				if (left != right)
					cout << " " << dan[right - left - 1];

			}
			left++;
		}

		if (left != num.length() && isPrint)
			cout << " " << dan[(num.length() - right - 1) / 4 + 2];

		right += 4;
	}

	return 0;
}
