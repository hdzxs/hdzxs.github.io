PATA_A_1039

Feb 24, 2020 #study

PAT_A_1039

Zhejiang University has 40000 students and provides 2500 courses. Now given the student name lists of all the courses, you are supposed to output the registered course list for each student who comes for a query.

Input Specification:

Each input file contains one test case. For each case, the first line contains 2 positive integers: N (≤40,000), the number of students who look for their course lists, and K (≤2,500), the total number of courses. Then the student name lists are given for the courses (numbered from 1 to K) in the following format: for each course i, first the course index i and the number of registered students *Ni* (≤200) are given in a line. Then in the next line, *Ni* student names are given. A student name consists of 3 capital English letters plus a one-digit number. Finally the last line contains the N names of students who come for a query. All the names and numbers in a line are separated by a space.

Output Specification:

For each test case, print your results in N lines. Each line corresponds to one student, in the following format: first print the student’s name, then the total number of registered courses of that student, and finally the indices of the courses in increasing order. The query results must be printed in the same order as input. All the data in a line must be separated by a space, with no extra space at the end of the line.

问题分析

问题给出n个学生，k门课，然后之后给出查询学生的名字，输出他选了哪些课，按照每门课的大小顺序输出。这题直接使用map进行对应后面会超时，所以选择对名字字符串进行hash，加快速度。另一方面，使用vector可以减少内存的使用，因为原题给出的课程数和学生数的乘积过大。

注意事项

  1. hash的实现方式，对于英文字母＋数字的hash可以使用如下方式实现，注意如果名字中还存在小写字母则改成52进制，即id*52。（注意id= 不是id+=。。。我越界了好久才找到问题所在）

    //哈希名字，减少存储空间
int getId(char name[]) {
	int id = 0;

	for (int i = 0; i < 3; i++)
		id = id * 26 + (name[i] - 'A');
	id = id * 10 + (name[3] - '0');

	return id;
}

  2. 使用cin/cout会超时所以使用print和scanf。

  3. 注意hash之后学生数组的大小，要能够容纳所有的可能，即26 * 26 * 26 * 10。

代码实现

#include<iostream>
#include<vector>
#include<cstdio>
#include<algorithm>
#include<string>
using namespace std;

const int  M = 26 * 26 * 26 * 10 + 1; //哈希的范围，学生散列数组需要这么大
vector<int> student_course[M];

//哈希名字，减少存储空间
int getId(char name[]) {
	int id = 0;

	for (int i = 0; i < 3; i++)
		id = id * 26 + (name[i] - 'A');
	id = id * 10 + (name[3] - '0');

	return id;
}

int main() {
	//使用char数组代替，string
	char name[4];
	
	int n, m;//n查询学生数，m课程数
	cin >> n >> m;

	for (int i = 0; i < m; i++) {
		int course_num, course_stu; //course_num课程代号，stu课程人数
		cin >> course_num >> course_stu;

		for (int j = 0; j < course_stu; j++) {

			scanf_s("%s", name,5);
			
			int id = getId(name);
			student_course[id].push_back(course_num);  //把这门课程添加到该学生的课程中
		}
	}

	for (int i = 0; i < n; i++) {
	
		scanf_s("%s", name, 5);

		int id = getId(name);

		sort(student_course[id].begin(), student_course[id].end());

		//输出学生姓名和选课数
		printf("%s %d",name, student_course[id].size());

		//输出学生课程号
		for (vector<int>::iterator it = student_course[id].begin(); it != student_course[id].end(); it++)
			printf(" %d", *it);

	/*	for (int j = 0; j < student_course[id].size(); j++)
			printf(" %d", student_course[id][j]);*/

		printf("\n");
	}

	return 0;
}
