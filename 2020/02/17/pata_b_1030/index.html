Feb 17, 2020 #study

PAT_B_1030

给定一个正整数数列，和正整数 *p*，设这个数列中的最大值是 *M*，最小值是 *m*，如果 *M*≤*m**p*，则称这个数列是完美数列。

现在给定参数 p 和一些正整数，请你从中选择尽可能多的数构成一个完美数列。

输入格式：

输入第一行给出两个正整数 N 和 *p*，其中 *N*（≤105）是输入的正整数的个数，*p*（≤109）是给定的参数。第二行给出 N 个正整数，每个数不超过 109。

输出格式：

在一行中输出最多可以选择多少个数可以用它们组成一个完美数列。

输入样例：

10 8
2 3 20 4 5 1 6 7 8 9

输出样例：

8

### 问题解析

首先可以很直观的想象完美数列一定是由递增的数列截取出来的，那么我们需要做的就是从0开始，对每一个i计算它的最长完美数列，再取其中最长的一个。而对于最长完美数列我们可以采用二分的方法来查找，即寻找第一个不满足条件的j。

注意事项

  1. 可能输入的数字规模大于int的范围，所以数组不能是int型。
  2. 注意每次lefg right的起始值，要包含i之后所有的元素。
  3. 如果所有数都满足条件，可以通过特判的方式直接得出，即if（a[n - 1] > p*a[i]）return n - 1

代码实现

#include<iostream>
#include<algorithm>
using namespace std;

int main() {
	int n, p, ans = 1;
	double num[100000];
	cin >> n >> p;

	for (int i = 0; i < n; i++) {
		cin >> num[i];
	}

	sort(num, num + n);

	for (int i = 0; i < n; i++) {//对于每一个i求出它的完美数列的长度
		int left = i + 1;               //注意起点和终点，起点i+1
		int right = n;
		int mid = 0;

		while (left < right) {             //left== right时退出
			mid = (left + right) / 2;
			if (num[mid] <= p * num[i]) {
				left = mid + 1;
			}
			else
				right = mid;
		}
		if (ans < left - 1 - i + 1)
			ans = left - i;
	}

	cout << ans;
}
