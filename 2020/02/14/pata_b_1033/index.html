1033 旧键盘打字 (20分)

旧键盘上坏了几个键，于是在敲一段文字的时候，对应的字符就不会出现。现在给出应该输入的一段文字、以及坏掉的那些键，打出的结果文字会是怎样？

输入格式：

输入在 2 行中分别给出坏掉的那些键、以及应该输入的文字。其中对应英文字母的坏键以大写给出；每段文字是不超过 105 个字符的串。可用的字符包括字母 [a-z, A-Z]、数字 0-9、以及下划线 _（代表空格）、,、.、-、+（代表上档键）。题目保证第 2 行输入的文字串非空。

注意：如果上档键坏掉了，那么大写的英文字母无法被打出。

输出格式：

在一行中输出能够被打出的结果文字。如果没有一个字符能被打出，则输出空行。

输入样例：

7+IE.
7_This_is_a_test.

输出样例：

_hs_s_a_tst

问题解析

思路就是建立一个hash_table，存储每个字母是否完好，注意‘+’如果损坏，大写字母不输出，而不是输出小写。

注意事项

  1. ‘+’要额外判断
  2. 有一个测试点可能是没有错误，即第一行为空，这种情况下，cin会出现问题，所以使用getline函数。（越来越觉得scanf和printf也不错。。。）

代码实现

#include<iostream>
#include<string>
#include<string.h>

using namespace std;

int main() {
	string wrong, in, out;
	bool hash_table[128];         //表示哪些字符是可以输出的
	
	memset(hash_table, true, sizeof(hash_table));

	//如果键盘没坏，第一行为空，使用cin输入存在问题，所以使用getline输入
	getline(cin, wrong);
	getline(cin, in);

	for (char i : wrong) {
		hash_table[i] = false;
		if (i >= 'A' && i <= 'Z')           //如果说是一个大写字母，则大小写均要改变
			i = i - 'A' + 'a';
			hash_table[i] = false;
	}

	for (int i = 0; i < in.length(); i++) {
		if (in[i] >= 'A' && in[i] <= 'Z') {
			if (hash_table['+'] == true && hash_table[in[i]] == true)
				cout << in[i];
		}
		else if (hash_table[in[i]] == true)
			cout << in[i];
	}
	cout << endl;
	return 0;
}
